Overview

The diagram below shows the packages which implement VGLViewer. Links to documentation for these packages are listed at the end of this page.

Package vglDocument implements a VGL document model. Inside this package is a subpackage, utils, which has utility classes for building documents and loading them from VGL XML. The utils package uses the third-party XML parser implemented in package nanoXML to interpret XML. [NANO]

The document model implemented in vglDocument incorporates a hierarchical bounding box algorithm implemented in package boundaryMap to assist in the fast retrieval of elements.

Package vglViewer implements the VGLViewer application itself. It contains a panel widget implemented in docViewPanel to view a VGL document model, and uses package sdaz to allow a user to navigate documents with speed-dependendent automatic zooming (SDAZ). [SDAZ]

The panel widget in package docViewPanel uses a renderer implemented in vglRenderer to render a VGL document model.

Documentation for each package is given below. You should read these in the order they are listed, since they are listed in order of dependency, working up towards the topmost dependendant package.
